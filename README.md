# 明日方舟公开招募（四星/五星快捷筛选标签）

访问：[https://1143910315.github.io/Arknights-Recruitment/](https://1143910315.github.io/Arknights-Recruitment/)

网页仅包含四星和五星干员的普通标签的公开招募，可以用于快速筛选招募。

---

## 功能特点

- **实时多选标签**：点击任意标签即可筛选出包含所有已选标签的招募组合。
- **智能发光提示**：
  - 紫色外光：当前标签可与已选标签构成四星组合。
  - 金色外光：当前标签可与已选标签构成五星组合（优先显示金色）。
- **灰色标签（不可用）**：表示该标签无法与当前已选标签构成任何四星或五星组合。点击灰色标签会清空其他已选标签，并只保留自身。
- **清空标签**：一键清除所有已选标签。
- **结果分组展示**：匹配的组合按五星、四星分组，并高亮已选中的标签。

## 使用方法

1. 打开网页。
2. 在标签区点击任意标签（可多选），下方结果区会实时更新匹配的招募组合。
3. 观察标签的发光颜色，可快速发现能形成四星/五星组合的“潜力标签”。
4. 若标签变为灰色，点击它会重置筛选为仅选中该标签。
5. 使用“清空标签”按钮可快速开始新的筛选。

## 数据说明

- 数据基于游戏内公开招募组合，仅包含四星和五星干员，并去除了时间锁、资深干员等特殊标签的组合。
- 所有标签均来自实际存在的招募组合，确保筛选结果的准确性。
