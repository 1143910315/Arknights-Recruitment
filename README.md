# 明日方舟公开招募 · 四星/五星快捷筛选

## 📌 项目简介
这是一个为《明日方舟》玩家设计的公开招募辅助工具，专注于 **四星与五星干员** 的标签组合筛选。页面仅包含四星和五星干员的普通标签数据，帮助博士们快速定位符合条件的招募组合，节省时间。

## 🌐 在线访问
👉 [点击立即使用](https://1143910315.github.io/Arknights-Recruitment/)  
（浏览器打开即用，无需安装）

## ✨ 功能特点
- **专注高星**：仅展示四星和五星干员的招募组合，排除三星干扰。
- **标签多选**：可自由组合任意标签，实时显示匹配的干员。
- **视觉反馈**：
  - **紫色光晕**：表示该标签可与当前选中构成四星组合。
  - **金色光晕**：表示该标签可与当前选中构成五星组合（优先级高于紫色）。
  - **灰色标签**：表示该标签无法与当前选中构成任何四星/五星组合，点击它会自动清空其他标签并只保留自身。
- **清晰分组**：匹配结果按五星、四星分组展示，组合卡片中高亮已选标签。

## 🖱️ 使用说明
1. 点击标签复选框选择您拥有的公开招募标签。
2. 下方结果区会实时列出包含所有已选标签的招募组合（按星级分组）。
3. 观察标签背景：
   - **紫色** → 再选上它即可凑出一个四星组合。
   - **金色** → 再选上它即可凑出一个五星组合。
4. 灰色标签无法与当前选中组合，但点击它会重置筛选为仅该标签自身。

## 📊 数据说明
- 数据基于游戏内公开招募组合整理，仅收录四星和五星干员。
- 组合中的干员名称及标签均与游戏一致。
- 若后续游戏版本更新，数据可能需要手动同步。
